Git Commands

  • git v (Used to check git version in our system)
  • git init (Used to initialise an empty git repository)
  • git add . (Used to upload all files)
  • git commit -m "Message related to what we have done" (Used to commit changes)
  • git remote origin "link from github" (Used to connect git and github)
  • git branch -M (Used to give branch name)
  • git push origin main (Used to push the files)
  • git status (Used to know the files are added or not)
  • git branch login (Used to give branch name as login)
  • git checkout login (navigate from one branch to another branch)
  • git status (Used to check status of branch name)
